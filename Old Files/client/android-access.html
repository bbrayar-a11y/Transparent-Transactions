Android Network Setup Guide:

  1. Same WiFi Network: Ensure both your phone and computer are connected to the same WiFi network
  2. Disable Mobile Data: Temporarily turn off mobile data to force WiFi connection
  3. Firewall Settings: Check that your computer firewall allows connections on port 3000
  4. Server Status: Verify the TrustLedger server is running on your computer
  5. IP Address: Use the IP address (192.168.1.4) not localhost
